# Agenda Inteligente

App mobile em React + Vite para usar como agenda pessoal estilo Foquinha.

## Recursos
- tarefas rápidas por texto
- metas semanais
- planejamento automático
- resumo para WhatsApp
- PWA instalável no celular
- anotações rápidas
- lembretes por notificação do navegador

## Rodar localmente
```bash
npm install
npm run dev
```

## Publicar na Vercel
1. Suba estes arquivos no repositório GitHub.
2. Entre na Vercel.
3. Importe o repositório.
4. Clique em Deploy.
